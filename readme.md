# Реализация тестового задания
```shell
# Небольшое резюме по (возможно) не самым понятным моментам в программе.
Все записи о переводах хранятся в PDA;
Чтобы начать отпралять лампорты необходимо создать PDA;
Если PDA не создан то в клиенте появится возможность его создать;
Тот кто его создаст будет записан в поле owner. То есть будет являтся влдаельцем и сможет выводить лампорты;
В PDA хранятся все успешные транзакции; 
В каждой транзкции есть поле withdrawn, оно маркирует состояние пожертвованных лампортов, то есть, если оно true 
то лампорты которые пожертвовал этот юзер - уже вывел владелец, в противном случае - лампорты пожертвованные в этой 
транзакции ещё не вывели и они хранятся на счету PDA.
```

```shell
# Для запуска клиента.
cd app
yarn start
```
